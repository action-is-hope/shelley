# Notification

<Source
  language="tsx"
  code={dedent`
    import { Notification } from "@actionishope/shelley";
  `}
/>

The Notification component allows you to display various types of notifications to users.
It can be used to show information, success messages, warnings, and alerts,
each with its respective icon and styling. The component can be closed by the
user if a close button is enabled. It also provides accessibility features
such as screen reader support.

### Custom Icon

It is possible to override the default icon but it must be wrapped in an Icon component
in order for it to work properly. Icon props can be passed to the Notification component and
they are infoIcon, successIcon, warningIcon, and alertIcon.

Displays an alert icon with the role of alert.
