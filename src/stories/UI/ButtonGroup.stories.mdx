# Button group

The ButtonGroup component can be used to group related buttons.
If Buttons created a cooperative it would look like a ButtonGroup.
Use with `Button`.

The buttons can be grouped by wrapping them with the `ButtonGroup` component. They need to be immediate children.

  <Story

    args={{
      variant: "primary",
    }}

</Preview>

## Split button

`ButtonGroup` can also be used to create a split button.

The dropdown could change the button text and action or be used to immediately trigger a related action.

<Preview>
  <Story name="e) Split button">
    <StoryCenteredInline>
      <ButtonGroup vol={2} tone={2} variant="primary">
        <Button>Publish</Button>
        <Button
          icon={
            <Icon alt="Select action">
              <path d="M8 12l-6.32-6.32 1.67-1.68 4.65 4.65 4.65-4.65 1.67 1.68-6.32 6.32z"></path>
            </Icon>
          }
          aria-label="Page actions"
          aria-haspopup="menu"
        />
      </ButtonGroup>
    </StoryCenteredInline>
  </Story>
</Preview>
